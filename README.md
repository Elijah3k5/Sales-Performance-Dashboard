# Sales Performance Dashboard


## Overview

This project presents an interactive Sales Performance Dashboard built in Microsoft Excel to analyze retail sales performance across regions, customer segments, product categories, and time. The dashboard helps identify business trends and supports data-driven decision-making through dynamic visualizations and KPI reporting.


## Business Problem

Retail managers require an easy-to-use dashboard to monitor sales performance and quickly answer questions such as:

- Which region generates the highest sales?
- Which product categories perform best?
- Which customer segments contribute the most revenue?
- How do sales change over time?
- Which products generate the highest sales?

This dashboard was designed to answer these questions using interactive Excel features.

## Dashboard Features

- Interactive slicers
  - Year
  - Region
  - Category
  - Customer Segment

- KPI Cards
  - Total Sales
  - Total Orders
  - Unique Customers
  - Average Order Value
  - Average Sales per Customer

- Visualizations
  - Sales by Region
  - Sales by Category
  - Sales by Customer Segment
  - Monthly Sales Trend
  - Top 10 Products by Sales

## Data Preparation

The dataset was prepared by:

- Checking for missing values
- Identifying duplicate records
- Validating data types
- Creating helper columns:
  - Year
  - Month Number
  - Month Name
  - Quarter
  - Year-Month


## Key Insights

- The West region generated the highest total sales.
- Technology was the highest-performing product category.
- Consumer customers contributed the largest share of revenue.
- Sales generally increased over the four-year reporting period.
- A small number of products generated a significant proportion of total sales.
